# Accessible Portfolio Website

A multi-page personal portfolio website built with semantic HTML5 and accessibility-first structure.

## Pages

- `index.html` - homepage
- `about.html` - profile and background
- `projects.html` - selected work
- `contact.html` - accessible contact form

## Accessibility Features

- Semantic landmarks: `header`, `nav`, `main`, `section`, `article`, and `footer`
- Skip link for keyboard users
- Explicit form labels and helpful descriptions
- Visible focus states
- ARIA labels and `aria-current` navigation state
- SEO-friendly titles, descriptions, canonical links, and social metadata

Open `index.html` in a browser to view the site.
